How should the mascot be built?

All 3 use pink & black, have a cursor-following eye, and blink. The difference is feel and fidelity.

A — SVG Vector

Hand-crafted SVG shapes. Clean lines with grungy stroke dashes and drip details. Eye is an SVG element that moves. Scalable, crisp at any size.

B — SVG + Noise Filters

SVG with feTurbulence/feDisplacementMap filters to distort edges and add ink-bleed grit. Closest to the hand-drawn original. More organic, slightly heavier to render.

C — CSS Shapes

Pure CSS divs and clip-paths. Fastest, simplest to animate. Less organic — more geometric/stylized. Good for a cleaner mascot vibe.
